Sudaka, el asistente operativo que vive en este perfil

Ademas del perfil publico, este GitHub Pages incluye una pagina dedicada a Sudaka, mi asistente operativo conectado a OpenClaw para GOUT Argentina.

Una sola conversacion para reportes, agenda, correos, Drive, GitHub y alertas.

Sudaka opera por Telegram, consulta fuentes conectadas, prepara respuestas accionables y mantiene reglas de seguridad para acciones sensibles. Su pagina publica explica que hace, como trabaja y que rutinas ya tiene activas.

Como construyo

Empiezo por el problema operativo, conecto fuentes reales y termino en una herramienta simple de usar.

01 · Fuente

Sheets, Drive, APIs, FTP/SFTP, Botmaker, Ads, Meta, OpenClaw o plataformas internas.

02 · Proceso

Limpieza, normalizacion, cache, validaciones, permisos y control de duplicados.

03 · Producto

Dashboard, agente IA, reporte automatico, alerta o flujo operativo.

04 · Operacion

Deploy, logs, monitoreo, GitHub Actions, GCP, alertas y mejora continua.
